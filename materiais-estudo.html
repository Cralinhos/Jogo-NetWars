Materiais para Estudo Netwars

Bem-vindo à seção de Materiais para Estudo do NetWars. Aqui você encontrará conteúdos complementares para reforçar seus conhecimentos em redes de computadores, com foco nos conceitos explorados durante o jogo. Use este espaço para revisar, aprender e dominar a teoria por trás das batalhas digitais.

Módulo 1 - As Redes de hoje

Neste módulo você irá conhecer os conceitos fundamentais das redes de computadores modernas. Aqui, você vai entender como as redes estão estruturadas atualmente, quais tecnologias são mais utilizadas e como elas impactam nosso dia a dia. É a base para compreender o cenário em que as batalhas do NetWars acontecem.

Módulo 2 - Switch Básico e configuração do dispositivo Final

Neste módulo, você irá aprender o funcionamento básico do switch, que conecta dispositivos em uma rede local usando endereços MAC, e como configurar um dispositivo final com IP, máscara de sub-rede, gateway e DNS. Esses ajustes permitem a comunicação correta na rede e são fundamentais para entender a lógica por trás das ações em NetWars.

Módulo 3 - Modelo OSI e TCP/IP

Este módulo apresenta os modelos de referência fundamentais para entender as redes de computadores: o modelo OSI (Open Systems Interconnection) com suas sete camadas e o modelo TCP/IP com suas quatro camadas. Você aprenderá como cada camada funciona, suas responsabilidades e como elas se comunicam entre si, formando a base para compreender todo o funcionamento das redes modernas.

Módulo 4: Camada Física

Este módulo explora a camada física do modelo OSI, responsável por transmitir bits por meio de sinais elétricos, ópticos ou de rádio. São abordados os principais tipos de meios de transmissão (cobre, fibra e sem fio), além de conceitos como codificação de sinais, largura de banda, atenuação e dispositivos como hubs e repetidores.

Módulo 5: Sistemas Numéricos

Este módulo aborda os principais sistemas de numeração utilizados em redes e computação: binário, decimal, hexadecimal e octal. Explica como converter entre esses sistemas e sua aplicação na representação de endereços IP, máscaras de sub-rede e outros elementos essenciais da tecnologia da informação.

Módulo 6: Camada de Enlace de Dados

Este módulo trata da segunda camada do modelo OSI, responsável pela entrega confiável de quadros entre dispositivos na mesma rede. Aborda o funcionamento de protocolos como Ethernet, controle de acesso ao meio (MAC), detecção de erros, endereçamento físico (endereços MAC) e o papel de switches na comunicação local.

Módulo 7: Comutação Ethernet

Este módulo explora o funcionamento da comutação Ethernet, tecnologia amplamente usada em redes locais. Aborda conceitos como tabelas de comutação (MAC address table), processo de encaminhamento de quadros, domínios de colisão e de broadcast, e o papel dos switches na segmentação de redes e no aumento de desempenho.

Módulo 8: Camada de Rede

Este módulo aborda a terceira camada do modelo OSI, responsável pelo endereçamento lógico e pelo roteamento de pacotes entre redes. São explorados conceitos como endereços IP, sub-redes, roteadores, tabelas de roteamento e os principais protocolos utilizados para garantir a entrega dos dados de forma eficiente em redes complexas.

Módulo 9: Resolução de Endereços

Este módulo trata dos processos usados para mapear endereços lógicos em endereços físicos dentro de uma rede. Aborda protocolos como ARP (Address Resolution Protocol) e ND (Neighbor Discovery), explicando como dispositivos encontram o endereço MAC correspondente a um endereço IP e como essa comunicação é essencial para o funcionamento das redes locais.

Módulo 10: Configuração Básica do Roteador

Este módulo ensina os conceitos fundamentais de configuração de roteadores. São abordadas tarefas como acesso ao modo de configuração, definição de hostname, senhas de acesso, configuração de interfaces, atribuição de endereços IP e salvamento das configurações. O objetivo é preparar o roteador para operar em uma rede básica.

Módulo 11: Endereçamento IPv4

Este módulo aborda o sistema de endereçamento IPv4, explicando sua estrutura, classes de endereços, máscaras de sub-rede e tipos de endereços (rede, host, broadcast). Também são tratados conceitos como subnetting, VLSM e a importância do planejamento de endereçamento para a organização e eficiência das redes.

Módulo 12: Endereçamento IPv6

Este módulo apresenta o protocolo IPv6, criado para substituir o IPv4 e atender à demanda por mais endereços IP. São abordadas a estrutura dos endereços IPv6, tipos de endereçamento (unicast, multicast e anycast), abreviações, escopos e prefixos. Também são discutidas as vantagens do IPv6 e seu papel na evolução das redes.

Módulo 13: ICMP

Este módulo aborda o protocolo ICMP (Internet Control Message Protocol), utilizado para enviar mensagens de controle e diagnóstico entre dispositivos de rede. São exploradas suas principais funções, como notificação de erros, teste de conectividade (comandos ping e traceroute), e o papel do ICMP na manutenção e solução de problemas em redes IP.

Módulo 14: Camada de Transporte

Este módulo explora a quarta camada do modelo OSI, responsável por garantir a entrega confiável dos dados entre aplicações. São abordados os principais protocolos da camada de transporte, como TCP e UDP, além de conceitos como controle de fluxo, controle de erros, portas de comunicação e multiplexação de dados.

Módulo 15 - Fundamentos de Segurança de Rede

Este módulo aborda os fundamentos essenciais de segurança de rede, incluindo conceitos de autenticação, autorização, criptografia e detecção de intrusões. São abordados protocolos de segurança como SSL/TLS, VPN, firewalls e sistemas de detecção de intrusões, além de práticas recomendadas para proteger redes contra ameaças cibernéticas.
